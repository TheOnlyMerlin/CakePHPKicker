Prerequisites

  1. Vagrant. See: http://www.vagrantup.com/downloads
  2. VirtualBox. See: virtualbox.org/wiki/Downloads
  3. (Highly recommended for windows users) Some kind of unix-like terminal for SSH in your command window. Suggested solution: install Git from http://msysgit.github.io/

Getting Started

If you want to get going right away, this will start your projoct with default settings right away.

  1. Navigate to the project directory
    • Protip: In windows press shift + right click in windows explorer in your folder to see an option in the right click menu to open command window in the current folder.
  2. Run command "vagrant up".
  3. Witness the magic.
    • It will take a few minutes to start. if it is your first time running vagrant, it will take more time to download the box for the first time.
  4. Go to the localhost project location (default localhost:8888) to see your project.

Confirm that the following match your configuration:

  1. Apache Root password set in /modules/manifests/init.pp
  2. Local host port is set to something you like (default: 8888) in Vagrantfile
